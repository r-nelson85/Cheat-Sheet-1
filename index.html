HTML Tag Reference

10 Basic Tags

Tag Name Description
<html> HTML HyperText Markup Language: Defines the root of an HTML document
<head> Head A container for the head elements
<title> Title Contains the title of the page
<body> Body Defines the document's body
<h1 - h6> Section headings 1 to 6 Defines HTML headings
<p> Paragraph Creates paragraphs of text
<a> A link Lets you create links to other content
<img> An image Lets you insert an image into web pages
<div> DIV A block level container used to add more structure to page content
<span> Span Inline element to add structure to content
